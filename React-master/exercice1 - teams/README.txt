index.html présente un "composant Team" statique (HTML/CSS).

Proposez un composant React Team équivalent mais où tout le contenu sera modifiable dynamiquement.

Notez que :
    - le nombre de membres de l'équipe doit pouvoir raisonnablement varier (plusieurs "lignes" peuvent être utilisées)
    - il vous appartient de décider de découper éventuellement votre composant en plusieurs sous-composants.
