index.html présente un "composant Plan" statique (HTML/CSS).

Proposez un composant React Plan équivalent mais où tout le contenu sera modifiable dynamiquement.

Notez que :
    - le nombre de Plans doit pouvoir raisonnablement varier (plusieurs "lignes" peuvent être utilisées)
    - le nombre d'items des Plans doit pouvoir aussi raisonnablement varier.
    - il vous appartient de décider de découper éventuellement votre composant en plusieurs sous-composants.
